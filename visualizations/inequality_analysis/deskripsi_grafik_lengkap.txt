🎯 GAMBAR 5: ANALISIS REGRESI LOGISTIK MULTIVARIAT
--------------------------------------------------

📈 5.1 Pentingnya Variabel (Magnitude Koefisien)
   Deskripsi: Menunjukkan seberapa kuat pengaruh masing-masing variabel
              terhadap probabilitas diterima KIP Kuliah

   Interpretasi:
   • Sumbu X: Magnitude (nilai absolut) koefisien regresi logistik
   • Sumbu Y: Variabel-variabel yang dianalisis
   • Semakin panjang bar = semakin kuat pengaruhnya
   • Urutan dari atas: variabel paling berpengaruh

📈 5.2 Arah Pengaruh Variabel
   Deskripsi: Menunjukkan apakah variabel meningkatkan atau
              menurunkan peluang diterima

   Parameter Warna:
   • Hijau (positif): Meningkatkan peluang diterima
   • Merah (negatif): Menurunkan peluang diterima
   • Garis putus-putus hitam: Titik netral (koefisien = 0)

📈 5.3 Kalibrasi Model: Prediksi vs Aktual
   Deskripsi: Mengukur seberapa akurat prediksi probabilitas model

   Interpretasi:
   • Sumbu X: Probabilitas prediksi model (0-1)
   • Sumbu Y: Tingkat penerimaan aktual
   • Garis merah putus-putus: Prediksi sempurna
   • Titik mendekati garis = model terkalibrasi baik

📈 5.4 Odds Ratio dengan Signifikansi Statistik
   Deskripsi: Mengukur seberapa besar variabel mengubah odds
              diterima, dengan tingkat signifikansi statistik

   Parameter Odds Ratio:
   • OR > 1 (hijau): Meningkatkan odds diterima
   • OR < 1 (merah): Menurunkan odds diterima
   • OR = 1: Tidak ada pengaruh

   Tingkat Signifikansi:
   • *** : p < 0.001 (sangat signifikan)
   • ** : p < 0.01 (signifikan)
   • * : p < 0.05 (cukup signifikan)
   • (kosong): p ≥ 0.05 (tidak signifikan)

   Contoh Interpretasi:
   • OR = 1.5***: Meningkatkan odds 50%, sangat signifikan
   • OR = 0.7**: Menurunkan odds 30%, signifikan

--------------------------------------------------

   • Missing values dihandle sesuai konteks masing-masing variabel

3. Interpretasi:
   • Semua tingkat penerimaan dalam rentang 0-1 (0% - 100%)
   • Error bars menunjukkan 95% confidence interval
   • Ukuran sampel mempengaruhi reliability analisis
